# Zombie Mayhem

Kill or be eaten in the **Zombie Mayhem**
 
Game build with HTML5, JavaScript, jQuery, CSS3 and SCSS.

![zombie mayhem](/zombie-mayhem.jpg)


### Demo
https://eliorshalev.github.io/zombie-mayhem/


### Usage
```
$ npm install
$ bower install
$ gulp serve
```
